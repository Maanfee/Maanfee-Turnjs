# Maanfee.Turnjs - Blazor Flipbook Component

A powerful and interactive Blazor flipbook component built with Turn.js, providing a realistic page-turning experience for digital publications.

## Features

- 🎯 **Realistic Page Turning** - Smooth page flip animations with hardware acceleration
- 📱 **Responsive Design** - Adapts to different screen sizes and orientations
- 🔄 **RTL/LTR Support** - Full support for both right-to-left and left-to-right languages
- 🔍 **Zoom Functionality** - Interactive zoom in/out with mouse and touch gestures
- ⌨️ **Keyboard Navigation** - Navigate using arrow keys
- 🖨️ **Print Support** - Built-in printing capabilities
- 🎨 **Customizable Templates** - Flexible header, content, and footer templates
- ⚡ **Lazy Loading** - Efficient page content loading
- 🎭 **Multiple Page Types** - Support for covers and content pages
